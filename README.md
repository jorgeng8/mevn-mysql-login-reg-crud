## Setup

instalar npm `npm install`en la carpeta principal
y en la subcarpeta 'cd ./client' tambien (la carpeta node_module es muy pesada)

npm run dev en la carpeta principal pra iniciar servidor backend nodemon

npm run dev en 'cd ./client' para inicar servidor de desarrollo frontend vue.js
